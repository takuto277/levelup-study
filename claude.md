# Antigravity (Claude) 向け全体ガイドライン及びコンテキスト (claude.md)

このファイルは、AIアシスタント（ClaudeやAntigravityなど）がこのリポジトリで作業する際に参照するべきグローバルなコンテキスト、ガイドライン、およびルールを含んでいます。

## リポジトリのコンテキスト
- **プロジェクト名:** LevelUp Study (勉強RPGアプリ) 
- **タイプ:** モノレポ (Monorepo)
- **目的:** iOSとAndroidの双方で展開される、ゲーム化された勉強アプリケーションです。ユーザーは勉強時間に応じてガチャ（キャラクターや武器）を引き、獲得したキャラクターを使用して「冒険」に出ることができます。

## 技術スタック (Technology Stack)
- **フロントエンド (モバイル):** `apps/`
  - iOS: Swift / SwiftUI
  - Android: Kotlin / Jetpack Compose
  - 共通ロジック (Shared Logic): ビジネスロジック、ネットワーク通信、状態管理など大部分の共有部分には Kotlin Multiplatform (KMP) を使用します。
- **バックエンド (Backend):** `backend/`
  - 言語: Go

## `claude/rules/` の使用について
具体的なアーキテクチャやコーディングのルールは `claude/rules/` ディレクトリに配置されています。AIアシスタントはこれらに必ず従ってください。

## Antigravity (AI) への一般指示
1. 新しい機能の実装を開始する前に、必ず `docs/` にある既存のドキュメント（要件定義・設計・提案書）を確認してください。
2. 新機能はクロスプラットフォームを前提として設計し、KMPを介したコード共有を最大化してください。
3. バックエンドのGo言語のコードは、Goらしい（Idiomatic Go）ガイドラインに従って実装してください。
